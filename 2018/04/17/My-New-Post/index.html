关于闭包，你必须知道这些

1.什么是闭包？

Like most modern programming languages, JavaScript uses lexical scoping. This means that functions are executed using the variable scope that was in effect when they were defined, not the variable scope that is in effect when they are invoked. In order to implement lexical scoping, the internal state of a JavaScript function object must include not only the code of the function but also a reference to the current scope chain.This combination of a function object and a scope (a set of variable bindings) in which the function’s variables are resolved is called a closure in the computer science literature.
——引用自“javascript权威指南 第6版”

实现水平和垂直居中的几种方式

参考博客用css实现垂直水平居中的几种方法

1.用inline-block和vertical-align来实现居中

2.如果高度、宽度已知，用相对绝对定位和负边距实现上下左右居中

3.利用绝对定位来实现居中

4.如果高度宽度未知，使用table-cell，inline-block实现水平垂直居中

display属性用于设置元素的展现形式，其属性很多，主要包括none（不显示）、block、inline、inline-block、list-item（作为列表显示）、table（块级表格）、inline-table（行内表格）、table-cell（作为表格单元格），inherit(从父级继承)。这里我们将父元素定义成单元格的形式，然后给它设置 text-align: center;vertical-align: middle;即可实现垂直和水平居中。

5.使用css3中的transform来时实现水平垂直居中：适合高度宽度未知的情况

transform属性用于定义元素的转换，包括元素的2D或3D移动（translate(x,y)）、缩放（scale(x[,y]?) ）、旋转（rotate(angle)）等。这里，我们首先将元素距离窗口上边和左边50%的位置，此时再将元素向左向上平移元素宽度和高度的50%，从而元素便定位到了中间的位置。

6.使用flexbox实现水平垂直居中

实现一个宽度为100%，宽高比为16:9的矩形块。

这个题目有两种方法实现，一种是利用padding-top/bottom实现，另一种是利用CSS3 vw 单位实现。参考博客div等比缩放

1.利用CSS3 vw 单位实现

CSS3 中新增了一组相对于可视区域百分比的长度单位vw、vh、vmin、vmax。其中vw是相对于视口宽度百分比的单位，1vw = 1% viewport width，vh是相对于视口高度百分比的单位，1vh = 1% viewport height；vmin是相对当前视口宽高中较小的一个的百分比单位，同理 vmax是相对当前视口宽高中较大的一个的百分比单位。

2.设置垂直方向的padding撑开容器

因为margin/padding-top/bottom 的百分比是按照 width的值计算的，所以我们可以将padding-top/bottom与屏幕宽度（因为矩形框的宽度是100%）的宽度进行绑定。这样就能实现一个高宽比固定的矩形块。
